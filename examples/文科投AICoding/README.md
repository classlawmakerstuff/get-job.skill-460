## 关键手法

1. **加参赛/求职定位头**——开头一句话锁定方向
2. **把"做过的事"翻译成"产品能力"**——活动运营观察 → 产品需求洞察、数据建模、Demo
3. **新增一个真做出的项目当王牌**——用 Vibe Coding 真做一个可运行 Demo（EventEase）
4. **统一专业排版**——section 分隔线、时间右对齐（脚本可生成同款 docx）
